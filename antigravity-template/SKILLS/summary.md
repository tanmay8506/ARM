## 19. SkillUI (npxskillui)
- **Directory**: `C:\Users\lenovo\Desktop\arm-artistry\antigravity-template\SKILLS\npxskillui-main`
- **What it does**: A CLI tool that crawls any live website, local codebase, or public git repo and automatically reverse-engineers its complete design system. Outputs a ready-to-use skill folder containing `SKILL.md`, `DESIGN.md`, `CLAUDE.md`, color/spacing/typography JSON tokens, bundled Google Fonts, and — in `--mode ultra` — scroll screenshots, hover/focus interaction diffs, CSS animation keyframes, and DOM component fingerprints, all packaged into a `.skill` ZIP.
- **When to use**: Use when you want to extract and replicate the visual language of any existing website into a reusable AI skill. Ideal for cloning competitor UI, bootstrapping a design system from an existing product, or building a reference library.
- **How to implement**:
  ```bash
  npm install -g skillui

  # Extract design system from a URL (default static mode)
  skillui --url https://target-site.com

  # Full cinematic extraction (requires Playwright)
  npm install playwright && npx playwright install chromium
  skillui --url https://target-site.com --mode ultra

  # Scan a local Next.js project
  skillui --dir ./my-nextjs-app
  ```
  Open the generated output folder and ask the AI to build your UI — it reads `SKILL.md` automatically.

## 20. Playwright CLI
- **Directory**: `C:\Users\lenovo\Desktop\arm-artistry\antigravity-template\SKILLS\playwright-cli-main`
- **What it does**: A token-efficient, CLI-based browser automation tool for coding agents. Provides a comprehensive command set (`open`, `goto`, `click`, `fill`, `screenshot`, `snapshot`, `eval`, `route`, `cookie-set`, `video-start`, tab management, storage management, etc.) to control a real Playwright browser session from the command line without pushing large accessibility trees into the model context. Far more token-efficient than Playwright MCP for agents working alongside large codebases.
- **When to use**: Use for end-to-end testing, capturing screenshots, mocking network requests, managing browser state (cookies/localStorage/sessionStorage), recording session videos, or any browser automation task within an agent workflow.
- **How to implement**:
  ```bash
  npm install -g @playwright/cli@latest

  # Install agent skills so the AI auto-discovers commands
  playwright-cli install --skills

  # Common usage
  playwright-cli open https://your-site.com --headed
  playwright-cli snapshot                    # capture page state for element refs
  playwright-cli click e15                   # click element by snapshot ref
  playwright-cli fill e20 "text" --submit    # fill field and press Enter
  playwright-cli screenshot                  # take a screenshot
  playwright-cli route "**/*.json" --body="{}" # mock a network request
  ```
  Detailed task guides live in: `playwright-cli-main/skills/`

## 23. Impeccable
- **Directory**: `C:\Users\lenovo\Desktop\arm-artistry\antigravity-template\SKILLS\impeccable-main`
- **What it does**: A comprehensive frontend design skill with a shared vocabulary of **23 commands** and **7 domain reference files** (typography, color-and-contrast, spatial-design, motion-design, interaction-design, responsive-design, ux-writing). Adds 27 deterministic anti-pattern rules + a 12-rule LLM critique pass. Key commands: `/impeccable craft` (full shape-then-build flow), `/impeccable audit` (a11y/perf/responsive checks), `/impeccable polish` (final pass, shipping readiness), `/impeccable critique` (UX design review), `/impeccable bolder`/`quieter` (amplify or tone down), `/impeccable animate` (purposeful motion), `/impeccable typeset` (fix font hierarchy), `/impeccable overdrive` (technically extraordinary effects), `/impeccable layout`, `/impeccable distill`, `/impeccable live` (iterate in browser). Also ships a standalone CLI (`npx impeccable detect`) that catches 24 design issues in HTML/CSS/URLs with no AI or API key required.
- **When to use**: Use for final polish, design reviews, typography fixes, motion design, accessibility audits, and shipping-readiness checks on any frontend project. Use the CLI to scan files or live URLs for anti-patterns without AI.
- **How to implement**:
  ```bash
  # Install (auto-detects AI harness: Cursor, Gemini CLI, etc.)
  npx impeccable skills install

  # Scan for anti-patterns without AI
  npx impeccable detect src/
  npx impeccable detect index.html
  npx impeccable detect https://your-site.com --fast --json
  ```
  Then use commands in AI chat:
  ```
  /impeccable audit the checkout page
  /impeccable polish the header
  /impeccable overdrive the hero section
  /impeccable critique landing
  ```
  Reference files: `impeccable-main/skill/reference/` | Full skill: `impeccable-main/skill/SKILL.src.md`
